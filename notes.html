HTML 5

HTML Elements

A HTML element is an individual component of a HTML document. A tag is a part of an element. It opens and closes an element apart from null elements that do not have closing tag.

Attributes in HTML elements are contained in the opening tag and never in the closing tag.

Each element in HTLM has an opening and closing tag, apart from null elements. Example: <em>Hello World!</em>

Null Elements

Elements that do not have a closing version of them e.g <br>, <hr>, <img> are called void or null elements.

Typgraphic Elements

Typographic elements tell the browser to how to render text. Here are a few examples:

  1. <b> Make text bold.
  2. <strong>Important text. This has a semantic value.
  3. <i>Make text italics
  4. <em>Emphasize the text. This has a semantic value.
  5. <mark>Highlight text.
  6. <small>Make text smaller.
  7. <del>Mark text as deleted.
  8. <ins>Mark text as inserted.
  9. <sup>Make the script a superscript.
  10. <sub>Make the script a subscript.

Block Elements

Block elements create an invisible box around the content they contain. They also add a a space above and below themselves. These elements always start on a new line and take up the full width available. This default formatting can of course be changed using CSS Block elements can only appear within a body element.
Examples of block elements

  • p
  • ol, ul
  • h1-h6
  • video
  • div
  • table
  • forms

Inline Elements

Inline Elements on the other hand, do not create the invisible box. They flow within the content. They also do not start on a new line.
Examples of inline elements

  • span
  • br
  • a
  • img

Lists

The p and li elements do not require closing tags. They are implied when the browser encounters another opening tag that cannot be nested inside the above elements.

CASCADING STYLE SHEETS (CSS)

CSS declration contains two parts: a selector and a declaration

Types of CSS selectors:

  • Type Selectors-This selector targets all the elements with the given name.
  • Class & Id Selectors-Class and id selector, will require you to add a class or id attribute to the html to use them.
    For class you need the . sign before the class name to tell CSS that this is a class selector. For id, you need the # sign before the id name to tell CSS that this is an id selector. An id can only be used to style a specific element. To name multiple html elements, use the class selector.

Measurements in CSS

There are two major kinds of lenghts in CSS

  1. Absolute Lengths
  2. Relative Lenghts

Absolute Lenghts

Absolute lenghs are based on actual physical unit. They will generally render the same across all devices. There may be some exceptions to this rule based on browser and OS settings.
The absolute lenghts are:

  • Pixel(px). A pixel is 1/96 of an inch. Due to the differences in screen resolutions, this may differ abit from screen to screen.
  • Centimetres(cm)- It is roughly 37.8 pixels.
  • Millimetres(mm)-It is roughly 3.78 pixels.
  • Inches(in)- It is roughly 96 pixels.
  • Points(pt)- It is roughly 1.3333 pixels
  • Picas(pc)- It is roughly 16 pixels.

Relative Lenghts

Relative lenghts are relative to another element's size or length.

  • em- It inherits the measurements based on the parent element.
  • rem- Rootem inherits its measurements based on the root of the document.
  • %-Relative to the parent size
  • View width(vw)- It is 1% of the width of the viewport.
  • View height(vh)- It is 1% of the height of the viewport
